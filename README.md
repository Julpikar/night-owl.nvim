# night-owl.nvim
Rewriting night-owl theme using lua.
## Install

    Plug 'Julpikar/night-owl.nvim'
    Plug 'nvim-treesitter/nvim-treesitter'

    

## Usage
    colorscheme 'night-owl'
 or:
 

    lua require('night-owl')
